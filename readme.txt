1. Install XAMPP in your computer
2. Put folder azkasir into /xampp/htdocs/azkasir
3. Open localhost/phpmyadmin in your browser
4. Create database with name 'azkasir'
5. Import 'azkasir.sql' in your database
6. Open azkasir application with typing localhost/azkasir
